# Aplicação criada com node.js para fins empíricos
## Estudo baseado na Playlist do Curso introdutório <a href="https://www.youtube.com/playlist?list=PLizN3WA8HR1w14FUaPYsP9q1nBHuTa1Dv">Curso de Node.js 2025 - Iniciante</a>
### Aula 1 - <a href="https://youtu.be/p9WpBHEXTms?si=KygDBwqlX6O_FW8U"> O Início da jornada</a>
Introdução ao curso e aos benefícios e comparativos do framework derivado do javascript e de sua escalabilidade de acessos em projetos, em que é aplicado predominantemente no Back-End.

### Aula 2 - <a href="https://youtu.be/p9WpBHEXTms?si=KygDBwqlX6O_FW8U"> Instalação no Windows </a>
Nessa aula tive o vislumbre dos processos de instalação sendo aplicados em conjunto com noções de especificações e tipografia das versões, como a explicação da versão LTS que é amplamnete utilizada pela comunidade com a finalizade da realização de testes. Além da instalação do _node.js_, relembrei as etapas da mesma ação com o pacote de bibliotecas do node, o _npm_

### Aula 3 - <a href="https://youtu.be/_5FGqCD6e9k?si=fkdm6Ahp9vCj9Fr4">Como baixar e instalar o VS Code </a>
Aula de tutorial de instalação e inicialização, junto de configurações básicas de como personalizar o ambiente de desenovlvimento

### Aula 4 - <a href="https://youtu.be/779zAF4RoYE?si=YwhXmY2qaKXAZ4gx"> Primeiro projeto Node com Express </a>
Aula que introduz a estruturação e criação do sevidor com node, utilizando da biblioteca <a href="https://expressjs.com/pt-br/">Express</a> que fornece recursos fundamentais para a criação do servidor de maneira eficiente(dentre os recursos estão: solicitações, respostas http, roteamento para criação de rotas), oferecedo a estrutura dorçal de funcionalidades e arquivos de configuração do _node_. Após o descarregamento da biblioteca por meio do terminal de comandos no VS Code, foi abordado como efetuar sua configuração inicial para o servidor.
