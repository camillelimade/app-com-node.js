# Aplicação criada com node.js para fins empíricos
## Estudo baseado na Playlist do Curso introdutório <a href="https://www.youtube.com/playlist?list=PLizN3WA8HR1w14FUaPYsP9q1nBHuTa1Dv">Curso de Node.js 2025 - Iniciante</a>

### Aula 1 - <a href="https://youtu.be/p9WpBHEXTms?si=KygDBwqlX6O_FW8U"> O Início da jornada</a>
Introdução ao curso e aos benefícios e comparativos do framework derivado do javascript e de sua escalabilidade de acessos em projetos, em que é aplicado predominantemente no Back-End.

### Aula 2 - <a href="https://youtu.be/p9WpBHEXTms?si=KygDBwqlX6O_FW8U"> Instalação no Windows </a>
Nessa aula tive o vislumbre dos processos de instalação sendo aplicados em conjunto com noções de especificações e tipografia das versões, como a explicação da versão LTS que é amplamente utilizada pela comunidade com a finalidade da realização de testes. Além da instalação do _node.js_, relembrei as etapas da mesma ação com o pacote de bibliotecas do node, o _npm_

### Aula 3 - <a href="https://youtu.be/_5FGqCD6e9k?si=fkdm6Ahp9vCj9Fr4">Como baixar e instalar o VS Code </a>
Aula de tutorial de instalação e inicialização, junto de configurações básicas de como personalizar o ambiente de desenvolvimento

### Aula 4 - <a href="https://youtu.be/779zAF4RoYE?si=YwhXmY2qaKXAZ4gx"> Primeiro projeto Node com Express </a>
Aula que introduz a estruturação e criação do servidor com node, utilizando da biblioteca <a href="https://expressjs.com/pt-br/">Express</a> que fornece recursos fundamentais para a criação do servidor de maneira eficiente(dentre os recursos estão: solicitações, respostas http, roteamento para criação de rotas), oferecendo a estrutura dorsal de funcionalidades e arquivos de configuração do _node_. Após o descarregamento da biblioteca por meio do terminal de comandos no VS Code, foi abordado como efetuar sua configuração inicial para o servidor.

### Aula 5 - <a href="https://youtu.be/pIbLLZy0Iho"> Rotas </a>
A aula aborda o conceito de rota principal e o funcionamento de rotas no node.js com express, exibindo a criação de rotas simples e dinâmicas com parâmetros para autenticação e como utilizar eles para o redirecionamento de requisições e personalização de respostas do site.

### Aula 6 - <a href="https://youtu.be/qy1X0c_4_-Q?si=J5VA0QvhmdzGxKf7"> Nodemon </a>
Essa aula se trata de um pequeno tutorial da instalação da biblioteca _Nodemon_, a qual permite a execução do servidor em paralelo com o código-fonte , possibilitando que as mudanças no código sejam atualizadas em tempo real no site.

### Aula 7 - <a href="https://youtu.be/UbhV0hD7_Wc?si=JA0k1n7e8DxS3hOE"> Instalação e configuração do MySQL </a>
A aula apresenta um tutorial detalhado e completo sobre a instalação e configuração do banco de dados MySQL, incluindo a preparação do ambiente e instalação de seu pacote de ferramentas (MySQL Server, Workbench, Router e Shell).
